## 🦅 1. LeBron🦅 - 채팅계의 르브론

### 채팅 스타일
```
12:26 LeBron 채님이
12:26 LeBron 일하러 가신다니
12:26 LeBron 뭔가
12:26 LeBron 안 어울리시네요
12:26 LeBron ㅋㅋㅋㅋㅋㅋㅋㅋ
```

**한 문장을 5개로 나눠서 보내는 스타일**. Enter 키를 마치 호흡하듯 누른다.

### 특징
- "아직 드랩 아닌가요?" 무한 반복 (드래프트 무새)
- 르브론 사랑 무한대 🏀
- "트리플 크라운 할 거임" 시즌 목표
- 술 먹고 압디아에 32불 지름 (전설의 드래프트)
- 트레이드 11회 참여, Jokić 영입

### NBA 비유: **Rob Pelinka (LAL GM)** 🟣🟡

르브론 팬 + 빅네임 수집 + 트레이드 많이 함 = 레이커스 GM 그 자체.
"우리가 조커 데려왔음!" 이러면서 결국 요키치 영입에 성공.

### MBTI: **ESFP (엔터테이너)**

외향적(E) - 채팅 최다
감각적(S) - 지금 당장 드래프트 원함
감정적(F) - 르브론 사랑
인식적(P) - 술 먹고 즉흥 드래프트

## 🥌 2. the beta_나는 돌맹이 - 트레이드 브로커

### 채팅 스타일
```
"트레이드 1회에 2000원씩 내게하면 대회 상금 많이 쌓이고 좋겠네 ㅋㅋㅋ"
"5만원에 트레이드 100% 확정권 팔면 어떻습니까? ㅋㅋㅋㅋ"
"안녕하세요 주식천재 베타입니다. 이익률 6% 넘겼습니다."
```

**트레이드 장사꾼 + 주식 자랑** 조합. 돈 이야기를 좋아한다.

### 특징
- 주식 수익 자랑 (매번 스크린샷 첨부)
- 트레이드 유료화 농담 (진심 반)
- Chai한테 폴조지 넘기고 시즌 접게 만든 전적
- Fox + Bane 영입 = 시즌 최고 강도 트레이드

### NBA 비유: **Sam Presti (OKC GM)** ⚡

드래프트픽 수집하듯 자산 모으기, 장기 투자 마인드, 그리고 "이거 나중에 비싸질 거야" 마인드.
Fox + Bane 영입이 바로 Presti식 장기 투자.

### MBTI: **INTJ (전략가)**

내향적(I) - 관찰 후 공격
직관적(N) - 미래 가치 예측
사고적(T) - 감정 배제, 숫자로 판단
판단적(J) - 계획적 트레이드

## 🍵 3. Chai - 요키치 원픽맨

### 채팅 스타일
```
"스테고님의 선수평가는 관심없습니다 ㅋㅋㅋ"
"당장 사과허ㅛㅔ욕"
"전 느바직관하는게 평생의 꿈인데 ㅋㅋ"
"올해는 요키치다. 요키치만 뽑고 나머지는 떨이듳록"
```

**드립 + 진심 + 요키치 사랑** 조합. 스테고와의 티키타카가 명물.

### 특징
- 요키치 원픽 ("130까지 지른다")
- 아들 미니바스켓 이야기 자주 등장
- 스테고한테 구라 + 장난 치기
- Flagg + Banchero 영입 (베스트 트레이드)

### NBA 비유: **Daryl Morey (PHI 사장)** 🔔

"요키치 데려오려면 얼마든 지불한다" = Morey의 Harden 영입 마인드.
확률과 데이터보다는 **원픽에 올인**하는 스타일.

### MBTI: **ENTP (변론가)**

외향적(E) - 수다 3위
직관적(N) - 미래 가치 (Flagg 영입)
사고적(T) - 냉정한 분석
인식적(P) - 유연한 전략

- "AI의 힘을 빌려봅니다" (분석 도구 활용)

### NBA 비유: **Pat Riley (MIA 사장)** 🔥

리그 총괄, 모든 것을 컨트롤하지만 직접 뛰지는 않음.
"형님이라도 안됩니다 ㅋㅋ" - 규칙은 규칙.

토론토 연결고리 + 분석적 접근 + 숨은 가치 발굴.
"켈엘이랑 매써린 기대합니다" - 슬리퍼픽 분석가.

- 반성문 스타일 ("작년에 내가 저랬구나")

### NBA 비유: **Sean Marks (BKN GM)** 🖤

조용히 팀 빌딩, 운이 안 따르는 GM의 대명사.
좋은 의도 + 나쁜 결과 = 베르가_A.

### MBTI: **ISFJ (수호자)**

내향적(I) - 조용한 참여
